# COVID-19 Data Dashboard (Canada Focus)

A dedicated monitoring dashboard providing a centralized view of pandemic statistics for Canada. This tool integrates real-time data visualizations from high-authority sources to track infection rates, healthcare impact, and vaccination progress.

---

## 🚀 Features

* **Multi-Metric Tracking:** Aggregates data on confirmed cases, hospitalizations, ICU occupancy, and cumulative deaths.
* **Live Data Integration:** Utilizes `iframes` to embed dynamic charts from *Our World in Data*, ensuring the information is always current based on their latest datasets.
* **Vaccination Progress:** Visualizes the percentage of the population vaccinated to track immunization milestones.
* **Comparative Analysis:** Provides a direct link to Wikipedia's global death rate comparisons for broader context.
* **Responsive Viewing:** The layout is designed to scale across different screen sizes using percentage-based widths.

---

## 🧠 Dashboard Components

The dashboard is structured to guide the user from general infection trends to specific healthcare system pressures:

1.  **Daily New Cases:** Tracks the velocity of the virus in Canada.
2.  **7-Day Rolling Average:** Smooths out reporting delays to show the true trend of transmission.
3.  **Vaccination Status:** Displays cumulative progress relative to the total population.
4.  **Healthcare Capacity:** Specific charts for current hospital patients and ICU admissions.
